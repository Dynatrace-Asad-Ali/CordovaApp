Set actionName for the load action

Agentconfig
Needed agent module configuration:
Support for XmlHttpRequest

The function actionName allows the change of an action name. Normally the name of the current action will get changed.
If you wan't to change a specific (not yet sent) action you can also pass the action id as second parameter.

In the example while the page loads a new actionName is set, which will set a custom actionName for the load action.

Code snippet
//This call will change the name of the load action, because this call happens while the page is still loading dtrum.actionName('Manual Load Action Name');
Example

If you call dtrum.actionName while the page is loading, you can rename the load action to your liking. In this example it will be called "Manual Load Action Name".
